FEATURES

为每天要交付的封面设计，不为演示截图设计。

01

业务 Prompt 先把图想清楚

把线路、客群、卖点、季节和价格整理成可执行的生图指令，少靠临场发挥。

02

自定义生图留给创意变化

需要更强风格控制时，直接输入完整 Prompt，生成更贴近活动调性的主视觉。

03

图片二次编辑不打断排版

选中画布图片继续改背景、风格或构图，结果回到当前图层，不重新开一套流程。

04

抠图修边服务最终画面

主体分割、手动修边和局部编辑都围绕“能不能马上排进封面”来做。

05

常用平台尺寸直接开画布

小红书、公众号、朋友圈和官网尺寸提前准备好，减少重复裁切和返工。

06

工程文件方便团队复用

工程导入导出、PNG/JPG/SVG 输出和倍率控制，让一次活动的版式下次还能用。

WORKFLOW

先定画幅，再处理主视觉，最后导出发布素材。

  1. 01 选择平台尺寸

    先确定小红书、公众号、朋友圈或官网画幅，别把裁切问题留到最后。

  2. 02 生成或导入主视觉

    用业务 Prompt、自定义 Prompt、素材库或本地上传拿到画面基础。

  3. 03 排版与图层编辑

    添加标题、Logo、形状和素材，精确处理每个对象的位置、层级和样式。

  4. 04 导出发布素材

    导出图片用于发布，保存工程用于复盘、改版和下一次活动。
