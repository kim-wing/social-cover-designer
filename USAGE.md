# YOUDESIGN 使用说明

YOUDESIGN 是一个本地桌面端社交封面设计工具，适合制作小红书封面、公众号首图、朋友圈长图封面、抖音/视频号/B 站等内容封面。核心能力包括尺寸预设、画布编辑、图片上传和粘贴、AI 生图 Prompt、OpenAI 生图、RMBG-1.4 抠图、手动蒙版修边、工程保存和图片导出。

## 1. 新建设计

打开软件后，左侧是素材和工具，中间是画布，右侧是属性面板。

常用步骤：

1. 在左侧选择平台尺寸预设。
2. 需要特殊尺寸时，在自定义宽高里输入像素值，再点击 `应用自定义尺寸`。
3. 点击 `新建空白画布` 可以清空当前内容并按当前尺寸重新开始。

当前支持常见社交媒体封面比例，也支持 100-8000 px 范围内的自定义画布。

## 2. 添加图片

可以用三种方式把图片放进画布：

1. 点击 `上传图片`，选择本地 JPG、PNG、WebP 等图片。
2. 从网页、截图工具、Finder 或其他软件复制图片后，回到 YOUDESIGN 直接粘贴。
3. 使用内置 AI 生图功能生成图片，生成完成后自动加入画布。

图片加入后会自动居中并被选中。选中图片后，可以在右侧调整位置、宽高、旋转、透明度、阴影、层级和背景色。

## 3. 智能抠图和手动修边

选中图片后，右侧会出现 `图片抠图` 区域。

### 智能抠图

点击 `智能抠图` 会使用 RMBG-1.4 模型自动去背景。

注意：

- 第一次使用需要联网下载模型。
- 用户不使用抠图功能时，不需要下载模型。
- 模型加载完成后，图片处理在本机完成。
- 下载模型和 OpenAI 生图是两个独立功能，抠图不需要 OpenAI API Key。

### 手动修边

点击 `修边` 进入蒙版编辑模式。即使没有先做智能抠图，也可以直接进入修边，软件会先创建一个全保留蒙版。

修边工具：

- `保留画笔`：把区域恢复为可见。
- `擦除画笔`：把区域擦除为透明。
- `画笔`：调整画笔直径，画布上会显示真实大小的圆形预览。
- `硬度`：控制边缘过渡，越高边缘越硬。
- `显示蒙版`：显示或隐藏蒙版覆盖层，方便看清修改范围。
- `撤销修边`：撤销上一笔。
- `完成修边`：退出修边模式并保留结果。
- `恢复原图`：放弃抠图或修边结果，恢复到原始图片。

## 4. 添加文字

左侧文字工具支持：

- `标题`：封面主标题。
- `文字`：普通说明文字。
- `副标题`：标题辅助信息。
- `大数字`：清单、攻略、步骤类封面常用。
- `竖排字`：边栏标题或装饰文案。
- `胶囊标签`、`角标`、`数字标签`：用于强调卖点、分类和序号。

选中文字后，可以在右侧修改内容、字体、字号、对齐、颜色、描边、透明度、阴影、位置、尺寸和旋转。

如果系统支持本地字体读取，可以点击 `读取本地字体` 使用电脑上的字体。字体授权需要用户自行确认。

## 5. 添加形状和品牌素材

左侧形状工具支持：

- `矩形`：色块、信息底板、按钮背景。
- `圆形`：装饰点、重点标识。
- `分割线`：标题下划线、信息分隔。
- `遮罩`：压暗背景或制造文字可读区域。

左侧还提供品牌 logo 和游小侠素材。点击素材即可加入画布，之后可以像普通图片一样调整大小、位置和层级。

## 6. 画布编辑

常用编辑操作：

- 单击对象：选中对象。
- 拖拽对象：移动位置。
- 拖拽控制点：缩放大小。
- 旋转控制：调整角度。
- `复制`：复制当前对象。
- `删除`：删除当前对象。
- `撤销` / `重做`：回退或恢复操作。
- 右侧 `左对齐`、`居中`、`右对齐`：快速对齐对象。
- 右侧层级按钮：上移、下移、置顶、置底。
- 缩放选择器：调整画布显示比例，不影响导出尺寸。

画布中的对象会显示在右侧图层列表，方便选择和调整层级。

## 7. AI Prompt 和 OpenAI 生图

左侧 AI 区域可以根据业务信息生成更完整的旅游封面 Prompt。

可填写内容包括：

- 标题
- 使用场景
- 受众
- 画面比例
- 风格
- 信息密度
- 卖点
- 出发地
- 目的地
- 天数
- 时间
- 价格
- 行程或景点
- 额外要求

生成后可以点击 `复制 Prompt` 到外部工具，也可以填写自己的 OpenAI API Key 后点击 `生成图片`。API Key 只保存在本机浏览器存储中。

## 8. 导出图片

顶部工具栏可以设置导出倍率和格式。

支持：

- PNG
- JPG
- 1x
- 2x

点击 `导出图片` 后，桌面版会打开系统保存窗口，可以选择文件夹和文件名。取消保存窗口不会导出，也不会弹出成功提示。

## 9. 导入和导出工程

当设计需要以后继续编辑时，点击 `导出工程` 保存 `.youdesign` 文件。桌面版同样会打开系统保存窗口，可以选择保存位置。

继续编辑时点击 `导入工程`，选择之前保存的 `.youdesign` 文件。导入工程会替换当前画布，导入前请确认当前设计是否需要先保存。

## 10. 更新软件

顶部工具栏提供：

- `检查更新`
- `重启安装`

检查到新版本后，下载完成可以通过重启安装更新。
